# LiteSpend Web Interface

Веб-интерфейс для приложения учета финансов LiteSpend.

## Технологии

- React 19
- TypeScript
- Vite
- TanStack Router
- Tailwind CSS
- Axios
- Zustand
- date-fns

## Установка

```bash
npm install
```

## Настройка

Создайте файл `.env` в корне проекта `web/`:

```
VITE_API_URL=http://localhost:8080/api/v1
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Функциональность

- Регистрация и авторизация пользователей
- Управление транзакциями (создание, редактирование, удаление, пагинация)
- Управление категориями (создание, редактирование, удаление)
- Статистика:
  - Текущий баланс (остаток, доходы, расходы)
  - Статистика по периодам (день, неделя, месяц)
  - Статистика по категориям
